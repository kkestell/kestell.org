## Implementation

### Parsing

The Ara compiler uses a handwritten LL(1) recursive descent parser.

### Code Generation

The Ara compiler emits LLVM IR.

## Source

Ara is written in C#. The source code is available on [GitHub](https://github.com/kkestell/ara).
